# Contributing

Thanks for helping kids learn with **truth-seeking** Super Tutor.

## Rules we won’t break

1. **Truth-seeking stays locked** (`core/truth-constitution.mjs`) — no “fun lies.”
2. **Grok only** (`core/provider-lock.mjs`) — no OpenAI/Anthropic/Ollama swaps.
3. **Educational safety** — family-friendly; no adult content.

## How to work

```bash
npm install
npm run check
npm start
```

## Updates via Grok Build

Maintainers can improve the agent with Grok Build / PRs.  
End users should pull releases — they should not need to edit `core/`.
